    - 用Matplotlib绘制折线图（展示随机生成的时间序列数据）和散点图（展示两组相关数据的分布），完成图形美化

- 成效要求：
        

    - 能独立完成Matplotlib基础绘图流程，熟练设置图形美化参数，理解子图布局逻辑
